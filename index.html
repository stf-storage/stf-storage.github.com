What is STF?

STF is a simple, yet very scalable distribute storage system.

STF powers sites like livedoor Blog, loctouch and many more, serving gigabytes after gigabytes of data. On one such system, it's serving 400Mbps of image at peak hours without a hitch.

GET STF!

STF is available from github. You will also need an HTTP server capable of handling X-Reproxy-URL, a mysql server, and backend storage server(s). Read the setup guide for more details
